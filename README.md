# DST The Forge items pack project
Don't Starve Together Mod collaboration project

Hello developpers,
The Forge event starts at the beginning of November and ends on December 4, 2017. This mod aims to make every The Forge items compatible for normal DST worlds.

You can get the code and help us to make it.

The current mod can be used in your DST world, all craft are listed below:

**Avancement**
=======
You can contribute to this project, change recipe, add assets etc ...

**Done:**
- Imports all weapons
- Imports all armors
- Imports all hats
- Recipes for weapons
- Recipes for armors
- Recipes for hats
- Strings for recipes
- New recipe tab
- Reticule placement for weapons

You an craft all items and use all weapons without special attacks

**Todo:**

- Assets for equiped book_fossil

- Assets for equiped armors

- aoeweapon_leap component for spear_lance and hammer_mjolnir (used in playercontroller.lua)
